# 智学平台页面结构设计 (Quantum Scholar Edition)

> [!abstract] 设计语言：Quantum Scholar
> 本设计采用 **极简主义 (Minimalism)** 与 **BENTO 布局**。通过主色 `Indigo (#4F46E5)` 与次色 `Cyan (#06B6D4)` 的渐变融合，营造权威且具未来感的 AI 学习氛围。

---

## 🎨 核心 UI 规范

| 元素 | 规范 |
| :--- | :--- |
| **布局风格** | BENTO Grid (非对称网格) |
| **视觉效果** | 玻璃拟态 (Backdrop Blur: 12px) |
| **主要字体** | Outfit (标题), Inter (正文) |
| **按钮交互** | `scale-95` 点击反馈, `shadow-lg` 悬浮反馈 |

## 🛠️ 交互微动效规范

1. **页面切换**: 使用 `Fade In` 与 `Horizontal Slide` 的组合（300ms）。
2. **列表进入**: 采用 **Staggered Entrance (交错进入)**，每项间隔 50ms。
3. **按钮反馈**: `Primary Button` 点击时会产生向外扩散的波纹动效。
4. **骨架屏**: 加载过程中显示具有 `Pulse` 效果的灰度 Bento 占位符。

---

## 📱 移动端适配 (Glassy Mobile)

- **导航**: 底部采用 iOS 风格的毛玻璃悬浮 Dock。
- **布局**: Bento 网格自动塌缩为 1 列，核心卡片全宽。
- **交互**: 侧边栏改为从底部弹出的 Drawer 抽屉。
